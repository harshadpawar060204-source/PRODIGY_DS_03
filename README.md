## Task 03 - Data Science Internship at Prodigy Infotech

### Objective
Build a Machine Learning model to predict whether a customer will purchase a product or service based on their demographic and behavioral data.

### Dataset
Bank Marketing Dataset

### Tools & Libraries Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded and explored the dataset using pandas.
2. Performed data preprocessing and handled missing values.
3. Converted categorical variables into numerical format.
4. Split the dataset into training and testing sets.
5. Built a Decision Tree Classifier model.
6. Evaluated the model using accuracy score.

### Output
- Trained machine learning model
- Prediction results on test data
- Model accuracy evaluation

### Conclusion
The Decision Tree model was able to classify whether a customer is likely to purchase a product based on the given features. This task helped in understanding machine learning workflows including preprocessing, training, and evaluation.
